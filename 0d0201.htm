Some words about DIP switches

Dip switches 2 and 3 on the host adapter board are to change the SCSI ID number of the drive. It goes as follows:



Tabelle 73: DIP switch settings
DIP2 DIP3 ID
ON ON 0
ON OFF 1
OFF ON 2
OFF OFF 3


Jumpers on your HD drive must be set allways to ID=0.
Any other ID will not be recognized by the Atari host adapter board (at all, ever).


Host Adapter:

Three DIP switches exist on a red block on the board.
The word 'ON' is printed on the front right of it as you look from the keyboard side of the machine. The DIP switches are marked 3, 2 and 1 (from left to right). All the DIP switches should point to the front of the board (ON) so that the board maps the SCSI ID to zero.

Flick the DIP switch marked 3 away from the front of the board and reboot. The internal drive will be remapped to SCSI ID 1. Return DIP switch 3 to the original position and flick switch 2 away from the front. Reboot and the drive will be ID 2. IF both 2 and 3 are flicked away from the ON position and you reboot, the drive will be remapped to SCSI ID 3. This is the maximum ID number available with the Atari host adapter board.

I don't know what the DIP switch marked 1 is for because it doesn't affect the SCSI ID number in my tests. However, I did hear once that it's related to running a second SCSI device from the internal host adapter. But, you need to remember that the power supply of the Mega STe is unlikely to be able to power a second drive, so I'd find out much more before I tried this.

English text from : Sid Celery UK Association of Atari User Groups



Harddisk Treiber

AHDI 3.01 aus dem Jahr 1989 so wie HDX 3.02 aus 1986,1987 oder 1988,1989.

Diese spezielle Version ist auch für die heute sicher nicht mehr im Einsatz befindlichen Orignal Atari-Festplatten SH104 und SH204 so wie SH205 gedacht. Der Treiber eignet sich auch direkt für die Megafile 20, Megafile 30 und Megafile 60 Festplatte und sollte verwendet werden wenn andere Tools versagen.

Sehr wichtig bei ALLEN DMA-Verbindungen ist ein äusserst kurzes DMA-Kabel, maximal 0.5 Meter, nie mehr, sofern es irgend wie machbar ist. Deutlich empfindlich ist Pin 12, Reset, aller dieser Leitungsverbindungen. Fängt man sich über diese Pin Störungen ein kann das gesammte Computersystem beeinträchtig werden.



Megafile Hostadapter

Originale Atari Megafile DMA Hostadapter generieren grundsätzlich kein SCSI Parity-Signal. Das hat Atariv verbockt. An allen angeschlossenen SCSI Festplatten muß Parity daher vor der Inbetriebnahme abgeschaltet werden.

Fehlermeldungen der Atari Controller werden über die LED in Blinkcodes mit einer Länge von 0.5 Sekunden ausgegeben.
