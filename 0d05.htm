13.5 Zweite Platte in ein HDplus-Subsystem


Einbau einer zweiten Platte in ein HDplus-Subsystem


Der Umbau ist nur möglich, wenn bereits eine 3 1/2"-Festplatte mit externem Controller eingebaut ist. Die zweite Festplatte muß die gleichen physikalischen Parameter (Cylinder, Köpfe, Codierung), wie die erste Festplattehaben. Die Parameter der verschiedenen HDplus-Subsystemen können der Tabelle entnommen werden.

Das bedeutet, man kann noch eine Platte reinsetzen und kommt so auf insgesamt 40 MB. Wird eine Platte mit mehr Köpfen, Zylindern eingebaut, können nicht u.U nicht genutzt werden.


Material:

- Stromversorgungskabel für zwei Festplatten
- 34 poliges Flachkabel mit zwei Festplatten-Anschlußsteckern
- 34 poliges Flachkabel für zweite Festplatte


- Verbindung Controller - Festplatte
Controller J2 => 20 poliges Flachkabel auf Festplatte 0
Controller J3 => 20 poliges Flachkabel auf Festplatte 1
Controller J1 => 34 poliges Flachkabel auf Festplatte 0 und 1 durchgeführt

- Festplatte 0 auf Laufwerk 0 jumpern
- Festplatte 1 auf Laufwerk 1 jumpern
- bei Festplatte 0 Abschlußwiderstände einschalten
- bei Festplatte 1 Abschlußwiderstände ausschalten 


 Blockschaltbild HDplus-Subsystem mit zwei Festplatten:

   |             |       | 34p.  20p.|    | 34p.   20p.|
   |Controller   |       |           |    |            |
    -------------        | Platte 1  |    | Platte 0   |
                          -----------      ------------



- auf Microboard ohne Festplattenabschaltung zusätzlich Jumper 5 setzen auf Microborad mit Festplattenabschaltung zusätzlich Jumper 8 setzen

- Adapteradresse für zweite Festplatte mit Dip-Switches 4 und 5 auf Microboard einstellen

- Factory-Init auf neuer Festplatte durchführen (Option HDPLUS.PRG/F-INIT/Drive 1)


Festplattenparameter, Jumperung Micrboard I,II HDplus-Subsysteme:

- Microboard I (M1): Microboard ohne Festplattenabschaltung
- Microboard II (M2): Microboard mit Festplattenabschaltung

M1: Jumperfeld auf Microboard I (ohen Festplattenabschaltung):
- bei zwei Festlatten zusätlich Jumper 5 stecken
- Factory-Init mit Jumper 6 durchführen

M2: Jumperfeld Microboard II, Jumper 9 zeigt auf Rückseite HD+
- bei zwei Festplatten zusätlich Jumper 8 stecken
- Factory-INIT mit Jumper 9 durchführen.
