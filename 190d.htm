25.13 Schaltplan


Atari Schaltpläne / Atari schematics



In diesem Abschnitt finden sie Links zu Schaltbildern aller Computer die von Atari gebaut wurden. Die Schaltpläne sind alle mit Scannern von verschiedenen Leuten digitalisiert und so umfangreich das sie dieser FAQ nicht beigelegt werden können.

This section holds some links to any ciruit diagram of any atari computer. All schematics are digitized from different people and the files are very large so that can't used in this archive.

Zum Herunterladen der Pläne benötigen sie ein Passwort das sie erhalten nachdem sie einen Textbeitrag zu dieser FAQ entrichtet haben.
