Jumper Array A (J1-J6):
All to bottom -> BNC (thin Ethernet)
All to top -> AUI (thick Ethernet)

 
Alle nach unten -> BNC (thin Ethernet)
Alle nach oben -> AUI (thick Ethernet)

 
Jumper D (J7)
To top -> BNC (Hier sollte kein Jumper stecken!)
To bottom -> AUI

 
Nach oben -> BNC (Hier sollte kein Jumper stecken!)
Nach unten -> AUI

 
Jumper Array C (J17-J23):
All closed -> TT
All open -> MegaSTE

 
Alle geschlossen -> Karte für Atari TT
Alle offen -> Karte für den MegaSTE

 
Jumper Array B (J8-J10)
Meaning unknown. Jumpering on mine TT-card: closed/open/closed

 
Bedeutung ist mir nicht bekannt.
Jumper auf meiner Karte: zu/offen/zu
