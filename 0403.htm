Abbildung 1 - TT Memory DIP switches


The MegTT can be fitted with 1, 4 or 16 MB Simms, you have always to use 4 simm together in one bank and of the ram same size. Otherway the board doasn't work well. The smallest size possible is 4 MB and the greatest memory size is 128 MB.

For each SIMM configuration used the dip switches must be correctly set according to the following diagram:

Atari TT-Ramkarten

Neuere TT-Ramkarten, z.B eine original Atari 16MB-Karte von DDD haben die Jumper etwas anders eingestellt, arbeiten aber idR so korrekt.

W101 kann auf s 1-2 stehen.
W102 kann auf s 1-2 stehen.

W103 auf s 2-3
W104 auf s 2-3 : Startadresse $01000000

W105 auf s 2-3 Burstmode für Ramzugriff

W106 auf s 1-2 16MB Ram

In dieser Einstellung sollten auch die neueren Karten perfekt arbeiten.
Weiter Informationen liegen mir z.Z nicht vor.

Wenn der Bildschirm mit einer neueren Karte jetzt deutliche Schlieren zeigt, sollen Sie versuchen zusätzliche Masseleitungen zu dieser Ramkarte zu führen. Das TT-Netzteil ist auf alle Fälle aussreichend.

Für den TT gab es ST-RAM-Erweiterungen mit bis zu 8 MB, insgesamt konnte man den Speicher so auf 10MB Ram erweitern. Das TT-RAM könnte theoretisch bis zu einem GB ausgebaut werden, ich weiß aber von keiner Erweiterung, die mehr als 256 MB hergibt. Ich hatte früher eine MightyMic mit der Maximalbestückung von 32 MB drin und bin dann auf die aixTT umgestiegen - die ließ sich bis zu 64 MB ausbauen und war um etwa 20% schneller. Die Magnum TT ist eine weiterentwickelte aixTT, sie ist auch schnell, kann aber bis zu 256 MB aufnehmen. Sie dürfte als einzige noch lieferbar sein.
