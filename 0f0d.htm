15.13 IDE Interface internal


IDE Interface and TOS 2.06 upgrade



The IDE Interface and TOS 2.06 upgrade board for the Atari ST(FM), STE and Mega ST computers enables up to two IDE hard disc or CD-ROM drives to be connected to the Atari computer, as well as providing and upgrade to the newest TOS version 2.06.



Picture 1 - The STE version of the interface




Picture 2 - The STFM version



Since the board includes plugs for both small 2.5" IDE drives, as well as ordinary 3.5" and 5.25" models, you can connect almost any IDE drive. If you have an ST or STE in its original case, a 2.5" hard disc drive is ideal since it can fit inside the case. If you have a Mega or a recased ST, then you might prefer to use a 3.5" or 5.25" mechanism. This also means that, with appropriate drivers (not supplied) you can connect an IDE CD-ROM drive as well as a hard disc drive.

The interface is compatible with almost all IDE hard disc drives, and since it is compatible with the Atari IDE interface as used in the ST Book and Falcon computers, ordinary hard disc driver software such as AHDI, CBHD and HDDRIVER can be used.

Installation in the ST(FM) and Mega requires soldering, which may mean that you need to get it installed by a professional. The STE version simply plugs into most STEs, although if your STE does not have a socketed CPU (rare) then soldering is required.

The IDE interface is known to be incompatible with Western Digital and IBM hard disc drives (although some driver software such as HDDRIVER has options that enable these drives to be used in a limited way). Any other IDE hard disc drive should be usable on the interface, however I obviously cannot test every device and I am unable to guarantee that any given device will be compatible with the interface.

Das IDE Interface und TOS 2.06 Erweiterungsboard für den Atari ST(FM), STE und die Mega ST Computers erlauben den Anschluß von zwei IDE Festplatten oder CD-ROM Laufwerken an ihrem Atari Computer, weiterhin steht mit dieser Erweiterung die für diese Computer aktuelle TOS Version V 2.06 zur Verfügung.



Picture 1 - The STE version of the interface




Picture 2 - The STFM version



Seit die Platine Steckanschlüsse für 2.5" IDE Festplatten und auch normale 3.5" und 5.25" Festplatten enthält, können sie an diesem Interface jede bekannte IDE Festplatte anschliessen. Wenn Sie einen ST or STE in einem Orignalgehäuse benutzen, ist eine 2.5" IDE Festplatte das ideale Laufwerk welches spielend mit in dieses Gehäuse zusätzlich eingebaut werden kann.

Wenn Sie einen MegaST oder einen umgebauten ST benutzen, bietet sich die Benutzung einer 3.5" oder einer 5.25" Festplatte an. Auch hier können IDE CD-ROM Laufwerke angeschlossen werden. Sie brauchen dazu einen Treiber der das Unterstützt. Ein Treiber ist dem Interface nicht beigefügt.

Das Interface ist kompatibel mit den meisten IDE Festplatten und es ist kompatibel zu dem IDE-Interface welches im ST-Book und im Falcon F030 eingebaut wurde. Festplattentreiber wie z,B AHDI, CBHD und HDDRIVER sind kompatibel und können daher benutzt werden.

Der Einbau des Interface in einen ST(FM) und einen MegaST sind mit einigen Lötarbeiten verbunden, was bedeutet das der Einbau von jemanden gemacht werden muß der sich damit gut auskennt. Die STE Version ist bei den meisten STes einfach zum einstecken. Hat die CPU in diesem Rechner keinen Sockel, das sind eigentlich nur sehr wenige Modelle, fallen auch hier Lötarbeiten an. Ansonsten nicht.

Dieses IDE Interface ist bekannt inkompatibel mit Western Digital und IBM Festplatten. (Möglicherweise können einige Platten, sofern der HDDriver verwendet wird, doch an diesem Interface betrieben werden. Aber das dürfte sich dann um Ausnahmen handeln).

Jede andere IDE Festplatte sollte mit diesem Interface benutzbar sein, bedenken sie jedoch bitte das ich niemals alle möglichen Festplatten selber testen kann. Ich kann daher auch eine Garantie dafür übernehmen das bereits vorhandene und ältere Festplatten problemlos betrieben werden können.
