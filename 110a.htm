17.10 Sharp / Atari


Datentransfer Sharp / Atari



Problem:

Ich würde gerne meinen Atari TT, unter Magic 6.x und meinen alten Sharp 1403H Taschenrechner miteinander verbinden. Geht das?

"SHARP-Archiv" scheint dafür die richtige Software zu sein. Dazu wäre dann aber ein neues, aber einfaches, Kabel herzustellen: 


                      Atari ST:                       Sharp:
                Druckerschnittstelle         11-polige Anschlußleiste

        +----------------------------------------+
        |                         +---[10k]----+ |    M  LS
        |                         | +-[10k]--+ | | -------------
        |    ---------------------|-|---     | | | |OOOOOOOOOOO|
        | 13 \O O O O O O O O O O O O O/ 01  | | | ---|--||-----
        |  25 \O O O O O O O O O O O O/ 14   | | +----+  ||
        |      --------------|--------       | +---------+|
        +--------------------+               +------------+


Es ist jeweils die Ansicht auf die Schnittstelle dargestellt.

Benötigtes Material:

1 Sub-D-Stecker 25-polig
1 Stiftleiste 11-polig
1 Kabel 3-polig
2 Widerstände 10k

Ich kann Dir leider keine Garantie dafür geben, dass der Transfer auch funktioniert, allerdings hatte ich damals von einem TT-Besitzer die Rückmeldung, dass der Transfer vom TT zum Sharp funktionieren würde. Die Übertragung vom Sharp zum TT hatte damals aber nicht funktioniert. Der User vermutete einen Kabelfehler, konnte aber keine weiteren Versuche starten, da er den nur geliehenen Sharp bereits wieder zurückgegeben hatte.

Unter 'http://www.tracht.onlinehome.de/site/software.htm' findest Du die Version 1.2.1 von 1995. Die Version ist noch unter GFA V2 erstellt und läuft auf dem TT nur mit einem 24-Bit Programm. Bei Bedarf kann ich Dir aber eine unter GFA V3 kompilierte Version schicken, die auch ohne Hilfsprogramm laufen sollte.
