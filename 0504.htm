5.4 Milan Audio Hardware


Allgemeines zu Audio am Milan


Audio-Erweiterungen

Zunächst wird es eine Anpassung von Standard - Soundblaster - Karten geben, so daß die Soundausgabe von Samples usw. realisiert werden kann. Vielleicht liegt dieser Kartenanpassung auch eine kleine HD-Recording-Software mit Zwei-Kanal-Ton bei. Im Zuge dessen soll es auch eine Betriebssystemerweiterung, vergleichbar mit dem in Atari-Kreisen bekannten Crazy-Sounds, geben.

Bereits erfolgreich an den Milan angepasst wurde die Startrack - Audiokarte, die professionelles HD-Recordung mit bis zu 96 kHz bietet. Mit dem Erscheinen des Milans ist auch eine MIDI-Schnittstellenkarte für den ISA-Port verfügbar. Diese extrem kostengüstige Lösung erweitert das System für nur DM 60.- um die Standardanschlüsse MidiIn und Midi Out. Der Milan setzt damit die erfolgreiche Tradtion der Atari-Computer auch im MIDI-Bereich fort. Etwa im Sommer wird das neue Milan-XBIOS erwartet. Das XBIOS ist derzeit für die Soundansteuerung im Falcon zuständig. Es wird soweit übernommen, daß es eine flexible Schnittstelle für Soundkarten aller Art zur Verfügung stellt, über die vergleichbare Funktionsaufrufe der Software an die jeweils angeschlossene Sound- bzw. Harddiscrecording - Hardware weitergegeben werden. Somit wird es möglich sein, mit einem vergleichbar geringen Aufwand bekannte HD-Recording-Karten an den Milan anzupassen. Sauber geschriebene Audio-Software dürfte auf diesen dann direkt laufen, bei anderen Programmen müssen nur geringe Anpassungen vorgenommen werden, um diese auf dem Milan zubetreiben.
