STBOOK info from atari's glossy flyers






Abbildung 1 - STBook, 1 MB Ram von innen



Features
8.5" x 11.4" footprint, 1.4" think
Weight 4.2 lbs (not certain which battery pack)
Up to 10 hours batter life (no idea, my battery is about shot I need to rebuild it.)
TOS in ROM (2.06 I think)
Built in transfer software.
Built in Vector Pad pressure sensitive pointing device (Best has replacement mylars for these still) I personally recommend an external mouse using a serial mouse driver.

 
Internal expansion port (modem would have went there if they had made one)
Internal 2.5" IDE hard drive

 
Specifications
CPU 8mhz 68000
RAM 1 Mb (did they make a 4mb) low power pseudo-static RAM
ROM 512k internal. includes TOS, file transfer and hard disk utilities

 
8mhz, 16 bit BliTTER chip

 
Keyboard 84 keys. (actually fairly nice) Full-size Atari Keyboard can be connected. (I would like to know how, no way possible that I know of, I think they may have been planning on doing this)

 
LCD 640x400 reflective Super-Twist Nematic (STN) (fragile and there are no more to be had if you break one)

 
Sound
Programmable sound generator with 3 voices. (NOT DMA SOUND - NOT STE Sound)

 
Expansion
120-pin full function expansion connector, allows full acces to the main CPU bus.
 
I/O
MIDI in
MIDI out (These are both through mini din connectors, BEST has Atari converters to take them to normal midi ports. Might be cheaper elsewhere)

 
RS232 (normal 9 pin)

 
Parallel port 8

 
ASCI port. (Not normal size, this uses a small cable that came supplied by atari.
Best still has some replacements of these)

 
Batteries
2 versions
1 case that holds 7 Alkaline 'AA' batteries
1 Rechrgeable NiCad batter pack, with AC adaptor. (I have one of these, anyone know a good place for purchasing the replacement cells?)

 
AC Adaptor
-(Delta)V recharge algorithm extends battery life. (optional) Rechareges in less than 2 hours, while computer is running.

 
Pointing Device
Atari pressure-sensitive VectorPad (tm), using built in Force Sensing Resistor technology. External electro-mechanical mouse can be connected.
 
Hard Disk
19 ms average access time. 40/80/120 Mbytes capacity.

 
Addendum
(From the ST Book User manual)

 
STBook has a connector for an external keyboard or numeric keypad. Atari does not manufacture these peripheral items, but suitable units may be available from your local computer retailer. In the near future, Atari will supply an optional external mouse, which wil attach to your STBook through this same connector. (Never happened that I know of)_

 
Graphics Resolution ST High (640x400)

 
Power Consumption 1.5 watts (average)
11 watts (maximum)

Mainboard-Reset

Akku und Netzteil entfernen. Danach die Sleep- und die Power-Taste für ca. 30 Sekunden drücken und halten. Nach dieser Zeit loslassen, Akku, Netzteil wieder anschliessen. Das STBook sollte wieder korrekt booten.

Im inneren des STBook befindet sich ein 3V-Akku, Hersteller laut Aufdruck ist Sony, jeder baugleiche Akku tut es aber auch. Ersetzt werden kann der Akku im Notfall durch eine normale Batterie 3V, mit einer Diode in Reihe, damit die Batterie vom Rechnernetzteil nicht geladen wird. Die Ersatzbatterie kann im leeren Modemfach, rechts ¦ber der Tastatur, untergebracht werden.
