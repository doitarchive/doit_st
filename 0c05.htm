Error Codes

E1 Pickup error
Hardware Error
 
E2 Focus error
Unable to read disc or no disc in tray
 
E3 GFS error
Abnormal CLV servo
 
E4 TOC read error
Wrong type of disc for autio command
 
E5 Data send error
Communication problem over the DMA channel
 
E6 Seek error
Search failed
 
E7 Medium error
Wrong type of disc in tray (audio or data)
 
d1 No disc
No disc in disc tray

CDAR504 Specifications

Standard
Sony und Philipps CD-Rom standard Mode1 and Mode2
 
Date Readout from Disc
153 kilobyte per second (continious)
 
Traverse Time
Less than 1.0 second (sequential move from innermost to outermost track)
 
Bit Error Rate
10^-12 (unscratched disc)
 
Interface
ACSI (two 19-pin connectors) Pinout upper
 
Audio Output
External: 1.4 V RMS 20 Ohm load Headphones: volume control
 
Frequency Response
20 to 20kHz +1/-6 decibels
 
Sound Noise Ratio
-75 decibels (limit -70 decibels)
 
Operating Temperatur
41 to 104 Grad F (5 to 40 Grad C) (noncondensing)
 
Humidity
20 to 80% (noncondensing)
 
AC Voltage
USA:117V (60Hz) EU: 220V (50Hz)
 
Power Consumption
USA: 0.15A EU: 0.08A
 
Installation
Horizontal
 
Loading
Front loading
 
Display
2 Digits
 
Remote Control
Audio control
 
Dimensions
2.85 in. (7cm)* 13in. (33cm) * 11.4in. (29cm)
 
Weight
9.5lb. (4.3kg)
